# Aura-RPG（俯视角游戏）



## 开发环境：

- UE5.2.1
- VS2022

## 最大RPC数量
默认两个有点不够用，参照lyra的10个

```cpp

#最大RPC更新数量
net.MaxRPCPerNetUpdate=10
```
